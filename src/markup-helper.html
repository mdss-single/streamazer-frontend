Модальные окна

  • Вызов модального окна

    Для вызова модального окона необходимо в атрибуте href указать id вызываемого окна, а также добавить класс js-modal для элемента.

    Если вызов окна идет не с ссылки, а с другого элемента (например с кнопки), тогда заместо атрибута href надо прописать атрибут data-src="#elementID"

  • Вызов окна регистрации

Пасхалка (easter egg)

Двойным кликом по шапке активируется турбо-режим полета

Основные элементы

.input-text 
<input type="text" name="" id="" class="input-text" placeholder="Input text">
<input type="email" name="" id="" class="input-text" placeholder="Input email">

При указании типа поля email на мобильном устройстве будет вызвана клавиатура для ввода email. Обычно она отличается только наличием символа @
